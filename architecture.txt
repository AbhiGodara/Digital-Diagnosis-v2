## Revised Architecture — Agentic AI Doctor (Current Approach)
```text
User Input (natural conversation via React UI)
            │
            ▼
┌───────────────────────────────┐
│       React Chat UI           │
└──────────┬────────────────────┘
           │
           ▼
┌───────────────────────────────┐
│     Langchain Agent           │
│   (Groq Llama-3.3-70b)        │
│                               │
│ Evaluates intent:             │
│ - General chat? -> direct ans │
│ - Symptoms? -> Extract & Tool │
└──────────┬────────────────────┘
           │ IF symptoms gathered
           ▼
┌───────────────────────────────┐
│   Tool: get_diagnosis()       │
│                               │
│ 1. Symptom Parser (LLM Call)  │
│    Extracts to 84-dim vector  │
│ 2. LightGBM Classifier        │
│    Predicts Top-3 Diseases    │
│ 3. Knowledge Base Lookup      │
│    Fetches medical advice     │
└──────────┬────────────────────┘
           │ Returns structured JSON
           ▼
┌───────────────────────────────┐
│     Langchain Agent           │
│   (Groq Llama-3.3-70b)        │
│                               │
│ Generates natural, empathetic │
│ response with diagnosis,      │
│ advice, and disclaimers.      │
└──────────┬────────────────────┘
           │
           ▼
┌───────────────────────────────┐
│       React Chat UI           │
└───────────────────────────────┘
```
