## HealthPilot Architecture

Workflow 1: Main Dashboard (Symptom Analysis)
```text
User Input (React UI Dashboard)
           │ (POST /api/diagnose)
           ▼
┌───────────────────────────────┐
│     Django Backend API        │
└──────────┬────────────────────┘
           │
           ▼
┌───────────────────────────────┐
│ 1. Symptom Parser (LLM Call)  │
│    Extracts to 84-dim vector  │
│ 2. LightGBM Classifier        │
│    Predicts Top-3 Diseases    │
│ 3. Knowledge Base Lookup      │
│    Fetches medical advice     │
│ 4. Summary LLM                │
│    Patient-friendly summary   │
└──────────┬────────────────────┘
           │ Returns structured JSON
           ▼
User Output (React UI Dashboard)
```

Workflow 2: AI Doctor Chatbot (Conversational)
```text
User Input (React Chat Widget)
           │ (POST /api/chat)
           ▼
┌───────────────────────────────┐
│     Django Backend API        │
└──────────┬────────────────────┘
           │
           ▼
┌───────────────────────────────┐
│     Langchain Agent           │
│   (Groq Llama-3.3-70b)        │
│                               │
│ Evaluates intent:             │
│ - General chat? -> direct ans │
│ - Symptoms? -> Extract & Tool │
└──────────┬────────────────────┘
           │ IF symptoms gathered
           ▼
┌───────────────────────────────┐
│   Tool: get_diagnosis()       │
│   (Reuses Workflow 1 ML pipe) │
└──────────┬────────────────────┘
           │ Returns tool result
           ▼
┌───────────────────────────────┐
│     Langchain Agent           │
│   (Groq Llama-3.3-70b)        │
│                               │
│ Generates natural, empathetic │
│ response with diagnosis,      │
│ advice, and disclaimers.      │
└──────────┬────────────────────┘
           │
           ▼
User Output (React Chat Widget)
```
